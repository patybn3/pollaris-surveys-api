# Pollaris
### Back-End

## Dependencies

- [API template](https://git.generalassemb.ly/ga-wdi-boston/express-api-template)


## Set Up

- `npm install`
- `npm install -g nodemon`
- `npm run server`
- From Browser, enter http://localhost:4741/examples to see if its working, it should say unauthorized.
